## 🚀 Features
*   **Arcade Aesthetic:** Retro-styled UI using custom fonts and animations to grab attention.
*   **Interactive UI:** Smooth item selection flows and engaging animations that increase user interaction (Click-Through Rate).
*   **Responsive Layout:** Fully optimized for mobile and desktop conversions.
*   **Developer Friendly:** Easy to integrate with your preferred backend, tracker, or affiliate network.
*   **Performance:** Lightweight and fast-loading code.

## 🛠 Tech Stack
*   **HTML5**
*   **CSS3** (with custom animations and variables)
*   **JavaScript** (for interactivity)
